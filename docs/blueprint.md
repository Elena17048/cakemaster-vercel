# **App Name**: Cake Canvas

## Core Features:

- Cake Showcase: Showcase cakes through a photo gallery, sortable by category.
- Cake Constructor: Interactive tool to guide users through customizing cake flavor, frosting, filling, size, shape, decorations, and text.
- Cake Visualization: Based on selections from the Cake Constructor, use AI tool to generate a preview image of the cake.
- Course Listing: Display available courses with details, prices, and booking options.
- Edible Printing Details: Provide service details for edible paper printing, including file submission guidelines, pricing, and contact information.
- About Us: Share the cake studio's story, team information, and values through a welcoming About Us page.

## Style Guidelines:

- Primary color: Soft cream (#F8F5ED) for a warm and inviting feel.
- Background color: Light beige (#FAF9F6), subtly desaturated to complement the primary.
- Accent color: Muted gold (#C2B280) to add a touch of elegance.
- Headline font: 'Playfair', a modern serif font for an elegant, fashionable feel. Body font: 'PT Sans' (sans-serif), suitable for readability in longer text.
- Use simple, elegant line icons in gold to represent cake categories, course features, and printing options.
- Maintain a clean and modern grid layout, using white space effectively to highlight cake images and important information.
- Incorporate subtle hover animations on cake images and course listings to provide a gentle and engaging user experience.
